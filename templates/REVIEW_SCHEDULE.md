# Review Schedule

Use retrieval, not rereading. A review asks the learner to produce an answer, solve a task, explain, or transfer.

## Review Rules

- If forgotten: schedule sooner and repair the misconception.
- If recalled with effort: keep interval modest.
- If recalled quickly and transferred: lengthen interval.
- If review workload is too high: compress reviews through mixed tasks.
